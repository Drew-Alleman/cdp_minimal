# cdp_minimal
 A minimal C++17/20 client library for the Chrome DevTools Protocol (CDP). Built for offensive security testing 

## Features
- Realtime Keylogging
- URL Tracking
- Redirection
- Web Cam Photos / List Web Cams
- Audio Capture / List audio devices
- Screenshotting
- Cookie Theft
